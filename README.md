# Para fazer alteracoes neste projecoes

### Baixe a pasta do [repositorio]

### Verifique que tem o `Python 3 ou superior` e o `pip` instalado

### Instale o Pandas
```
pip install python

```

### Instale o MatplotLib
```
pip install matplotlib

```

### Execute o arquivo `main.py`
```
python main.py

```
